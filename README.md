# GTA5RP Cargo Stats

**GTA5RP Cargo Stats** — это приложение для стримеров и игроков GTA5 RP, которое позволяет в реальном времени отслеживать статистику доставок грузов и отображать её в OBS через настраиваемый HTML-виджет.

## Основные возможности

- 📦 **Выбор грузовика** из предопределённого списка (с автоматической подгрузкой иконки).
- 📊 **Учёт выполненных заказов**: добавление EXP, прибыли, автоматическое увеличение счётчиков рейсов (за день, за месяц, всего).
- ⛽ **Учёт расхода на бензин**: сумма вычитается из чистой прибыли.
- 🔄 **Автоматический сброс** дневной и месячной статистики по московскому времени (даже если программа не была запущена в момент сброса).
- ✅ **Гибкое отображение** в виджете: можно включать/отключать показ строк «Всего рейсов», «Чистая прибыль», «Потрачено на бензин».
- 🎨 **Кастомизация виджета** через CSS (цвета, размеры).
- 🔊 **Звуковой эффект** монет при добавлении заказа.
- 🌐 **Интеграция с OBS** через локальный HTML-файл и Flask-сервер.
- 📱 **Поддержка Twitch-команд** (в разработке).

## Требования

- Python 3.8 или выше
- Windows (для звука используется `winsound`, на других ОС потребуется замена)
- OBS Studio (для отображения виджета)
