# GTA5RP Cargo Stats

**GTA5RP Cargo Stats** — это приложение для стримеров и игроков GTA5 RP, которое позволяет в реальном времени отслеживать статистику доставок грузов и отображать её в OBS через настраиваемый HTML-виджет.

# [⚡️ Загрузить для Windows](https://github.com/s3r4tv/cargo_stats/releases/latest) [📺 Стримеры с трекером](https://s3r4.tv/cargo-live)

![Скриншот](https://i.imgur.com/ocTjbzL.png)

## Основные возможности

- 📦 **Выбор грузовика** из каталога моделей с автоматической подстановкой иконки.
- 📊 **Учёт заказов** — начисление опыта и денег по заказу, автоматическое обновление счётчиков рейсов (день, месяц, всего).
- ⛽ **Бензин и ремонт** — отдельные суммы трат; вводимые суммы **уменьшают показатель общей прибыли с заказов** (он может уходить в минус и подсвечиваться в виджете), при этом **счётчики потраченного на бензин и на ремонт** копятся независимо.
- 🔄 **Автоматический сброс** дневной и месячной статистики по московскому времени, в том числе если программа в момент сброса не была запущена.
- ✅ **Гибкое отображение виджета** — можно включать и отключать строки «Всего рейсов», прибыль с заказов, расход на бензин и ремонт.
- 🎨 **Кастомизация виджета** через CSS (цвета, размеры, оформление).
- 🔊 **Звуковой сигнал** при добавлении: выполненного заказа, расхода на ремонт \ топливо, новой версии программы.
- 🌐 **Интеграция с OBS** — локальный HTML-виджет и Flask-сервер для обновления без ручного обновления источника.
- 💬 **Интеграция с Twitch** — команды в чате для просмотра статистики и учёта бензина/ремонта в связке с приложением.

## Требования

- Python 3.8 или выше (скачать можно с [официального сайта](https://python.org/downloads/)), если не запускается программа.
